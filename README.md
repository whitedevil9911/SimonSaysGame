## Overview

**Simon Says Game** is a fun, memory-based game where you need to repeat a sequence of colors in the same order they flash. The game keeps track of the number of levels you have completed, and each correct sequence adds another color to the pattern. The challenge increases with each level, and the game ends when you make a mistake.

### Game Rules:
1. The game will show you a sequence of colored buttons.
2. You need to click the buttons in the same order as the sequence.
3. Each time you get the sequence correct, the game will add another button to the sequence.
4. If you get the sequence wrong, the game will end and show your score.

## How to Play

1. **Press any key** to start the game.
2. **Watch the Sequence**: The game will flash a sequence of colored buttons one by one.
3. **Repeat the Sequence**: Click the buttons in the same order they flashed.
4. **Level Up**: After completing the sequence, the game adds another button to the sequence.
5. **Game Over**: If you press the wrong button, the game ends and shows your score. Press any key to restart.

## Features

- **Colorful Buttons**: Click on flashing colored buttons to repeat the sequence.
- **Increasing Difficulty**: Each time you get a sequence right, a new button is added to the sequence, making it harder.
- **Responsive Design**: Fully mobile and desktop compatible.
- **Game Over Screen**: After a mistake, it shows your score and gives you the option to restart.
- **Simple Gameplay**: Easy to learn, hard to master!
